# Configure AI Applications to Optimize Search Results (Challenge Lab)

This repository contains reference material for the Challenge Lab  
**Configure AI Applications to Optimize Search Results** (Lab ID: **GENAI091**).

## How to Use
1. **Open commands.md** in this folder and copy necessary prompt and use wherever as instructed in youtube video.  
2. **Complete the challenge tasks independently** based on the business scenario provided in the lab.  
3. This is an **intermediate-level challenge lab** that requires a **minimum score of 80%** to pass.  
4. The lab uses **temporary Google Cloud credentials**, and the timer cannot be paused once started.

## Lab Description
This challenge lab validates your ability to:
- Configure **search serving controls** in AI Applications  
- Boost and bury search results to improve relevance  
- Filter unwanted results from the search experience  
- Display **metadata** in the search interface  
- Verify configurations using the automated scoring system  

## Disclaimer
This repository is provided **for educational purposes only**.  
It is intended to support learning and understanding of **AI-powered search optimization** on Google Cloud.
