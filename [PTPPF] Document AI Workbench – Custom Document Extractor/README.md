# README — PTPPF: Document AI Workbench – Custom Document Extractor

This folder contains the code for the lab **[PTPPF]: Document AI Workbench – Custom Document Extractor** (Lab ID: **PTPPF040**).

## How to Use
1. **Download this folder** or Download The Image File,
2. Upload the image for schema generation process. 
3. This is an **advanced challenge lab** that requires an **80% passing score**.  
4. Once started, the lab timer cannot be paused. All work is done using **temporary Google Cloud credentials**.

## Lab Description
This PTPPF challenge lab requires you to create a **Custom Document Extractor** for new document types that do not have pre-trained processors.  
You will configure Workbench, annotate documents, train a custom model, and test extraction accuracy as part of the assessment.
