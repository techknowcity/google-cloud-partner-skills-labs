# Extend Gemini Enterprise Assistant Capabilities (Challenge Lab)

This repository contains reference material for the Challenge Lab  
**Extend Gemini Enterprise Assistant Capabilities** (Lab ID: **GENAI087**).

## How to Use
1. **Download this repository** or **copy the configurations/code** into your Google Cloud environment wherever needed.  
2. **Update all required variables** such as project ID, region, assistant configuration, and Cloud Run service URLs.  
3. **Implement and configure each command** based on the challenge scenario.  
4. This is an **advanced challenge lab** that requires a **minimum score of 80%** to pass.  
5. The lab uses **temporary Google Cloud credentials**, and the timer cannot be paused once started.

## Lab Description
This challenge lab tests your ability to:
- Extend a **Gemini Enterprise assistant** with custom capabilities  
- Add a **Gmail action** for email drafting and sending  
- Enable **grounding with Google Search**  
- Route queries to a **custom Conversational Agent**  
- Invoke backend logic via a **Cloud Run function**  
- Validate results using the automated scoring system  

## Disclaimer
This repository is provided **for educational purposes only**.  
It is intended to support learning and understanding of **Gemini Enterprise assistant extensibility** on Google Cloud.
