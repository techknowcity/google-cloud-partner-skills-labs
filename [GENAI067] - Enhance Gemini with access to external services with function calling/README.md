# Enhance Gemini with External Services using Function Calling (Challenge Lab)

This repository contains reference material for the Challenge Lab  
**Enhance Gemini with access to external services with function calling**  
(Lab ID: **GENAI067**).

## How to Use
1. **Download this repository** or **copy the code** with code.md into your **Vertex AI Workbench** environment. 
2. **Update all required variables** such as project ID, region, and API configuration if needed anywhere.  
3. This is an **intermediate-level challenge lab** that requires a **minimum score of 80%** to pass.  
4. The lab uses **temporary Google Cloud credentials**, and the timer cannot be paused once started.

## Lab Description
This challenge lab validates your ability to:
- Extend **Gemini** using **function calling**  
- Implement the **ReAct pattern** for LLM reasoning and actions  
- Integrate external systems such as databases and third-party APIs  
- Use **Vertex AI** and **Vertex AI Workbench** effectively  
- Verify task completion using automated scoring
